# 深入理解防抖（Debounce）

## 前言

在前端开发中，我们经常会遇到一些需要频繁触发的事件，比如滚动事件、输入框输入事件、 resize 事件等。如果直接处理这些事件，可能会导致性能问题。防抖（Debounce）就是一种常用的性能优化技术，它可以有效地减少事件的触发频率，提高应用的性能。

## 什么是防抖？

**防抖**是一种函数优化技术，它的核心思想是：**当事件被触发后，延迟一段时间再执行回调函数，如果在这段时间内事件再次被触发，则重新计时**。

简单来说，防抖就是让函数在一定时间内只执行一次，避免频繁执行导致的性能问题。

## 防抖的实现

### 基础实现

```javascript
function debounce(func, wait, immediate = false) {
  let timer = null; // 存储定时器
  return function (...args) {
    //保存上下文参数
    const context = this;
    // 如果有定时器，清除定时器
    if (timer) clearTimeout(timer);

    //立即执行的逻辑
    if (immediate) {
      //判断是否是第一次触发
      const callNow = !timer;
      timer = setTimeout(() => {
        timer = null;
      }, wait);
      //首次触发时立即执行
      if (callNow) func.apply(context, args);
    } else {
      //非立即执行：重新设置定时器，wait时间后执行函数
      timer = setTimeout(() => {
        func.apply(context, args);
      }, wait);
    }
  };
}
```

### 代码解析

1. **参数说明**：
   - `func`：需要防抖的函数
   - `wait`：等待时间，单位为毫秒
   - `immediate`：是否立即执行，默认为 false

2. **核心逻辑**：
   - 使用闭包保存定时器 `timer`
   - 每次触发事件时，清除之前的定时器
   - 如果 `immediate` 为 true，则在首次触发时立即执行函数
   - 如果 `immediate` 为 false，则在等待时间后执行函数

## 防抖的应用场景

### 1. 输入框搜索

当用户在输入框中输入内容时，我们通常会希望在用户停止输入后再发送搜索请求，而不是每次输入都发送请求。

```javascript
const searchInput = document.getElementById("search");

function handleSearch(query) {
  console.log("搜索:", query);
  // 发送搜索请求
}

// 使用防抖优化搜索
const debouncedSearch = debounce(handleSearch, 300);

searchInput.addEventListener("input", (e) => {
  debouncedSearch(e.target.value);
});
```

### 2. 滚动事件处理

滚动事件会频繁触发，使用防抖可以减少处理函数的执行次数。

```javascript
function handleScroll() {
  console.log("滚动位置:", window.scrollY);
  // 处理滚动逻辑
}

// 使用防抖优化滚动处理
const debouncedScroll = debounce(handleScroll, 200);

window.addEventListener("scroll", debouncedScroll);
```

### 3. 窗口 resize 事件

窗口大小改变时，resize 事件会频繁触发，使用防抖可以优化性能。

```javascript
function handleResize() {
  console.log("窗口大小:", window.innerWidth, window.innerHeight);
  // 处理窗口大小改变的逻辑
}

// 使用防抖优化 resize 处理
const debouncedResize = debounce(handleResize, 200);

window.addEventListener("resize", debouncedResize);
```

### 4. 按钮点击防重复提交

防止用户快速点击按钮导致的重复提交。

```javascript
const submitBtn = document.getElementById("submit");

function handleSubmit() {
  console.log("提交表单");
  // 提交表单逻辑
}

// 使用防抖防止重复提交
const debouncedSubmit = debounce(handleSubmit, 1000, true);

submitBtn.addEventListener("click", debouncedSubmit);
```

## 防抖与节流的对比

| 特性         | 防抖（Debounce）                                     | 节流（Throttle）                             |
| ------------ | ---------------------------------------------------- | -------------------------------------------- |
| **触发方式** | 事件触发后延迟执行，如果在延迟期间再次触发则重新计时 | 事件触发后立即执行，然后在一定时间内不再执行 |
| **执行频率** | 最后一次触发后的等待时间结束后执行一次               | 每隔一定时间执行一次                         |
| **应用场景** | 输入搜索、表单提交                                   | 滚动事件、游戏中的移动检测                   |
| **核心思想** | 合并多次操作，只执行最后一次                         | 限制执行频率，固定时间执行一次               |

## 防抖的高级实现

### 带取消功能的防抖

```javascript
function debounce(func, wait, immediate = false) {
  let timer = null;

  const debounced = function (...args) {
    const context = this;

    if (timer) clearTimeout(timer);

    if (immediate) {
      const callNow = !timer;
      timer = setTimeout(() => {
        timer = null;
      }, wait);
      if (callNow) func.apply(context, args);
    } else {
      timer = setTimeout(() => {
        func.apply(context, args);
      }, wait);
    }
  };

  // 添加取消方法
  debounced.cancel = function () {
    if (timer) {
      clearTimeout(timer);
      timer = null;
    }
  };

  return debounced;
}
```

### 使用示例

```javascript
const debouncedFunction = debounce(() => {
  console.log("执行");
}, 1000);

// 触发防抖函数
debouncedFunction();

// 取消防抖
setTimeout(() => {
  debouncedFunction.cancel();
}, 500);
```

## 防抖在框架中的应用

### Vue 中使用防抖

```vue
<template>
  <div>
    <input
      type="text"
      v-model="inputValue"
      @input="handleInput"
      placeholder="搜索..."
    />
  </div>
</template>

<script>
export default {
  data() {
    return {
      inputValue: "",
    };
  },
  methods: {
    // 使用防抖处理输入
    handleInput: debounce(function () {
      console.log("搜索:", this.inputValue);
      // 发送搜索请求
    }, 300),
  },
};
</script>
```

### React 中使用防抖

```jsx
import React, { useState, useCallback } from "react";

function SearchInput() {
  const [inputValue, setInputValue] = useState("");

  // 使用 useCallback 和防抖
  const handleSearch = useCallback(
    debounce((query) => {
      console.log("搜索:", query);
      // 发送搜索请求
    }, 300),
    [],
  );

  const handleInput = (e) => {
    const value = e.target.value;
    setInputValue(value);
    handleSearch(value);
  };

  return (
    <input
      type="text"
      value={inputValue}
      onChange={handleInput}
      placeholder="搜索..."
    />
  );
}
```

## 性能优化建议

1. **合理设置等待时间**：根据具体场景设置合适的等待时间，太短起不到防抖效果，太长会影响用户体验。

2. **使用立即执行模式**：对于需要立即反馈的场景，使用 `immediate: true`。

3. **结合节流使用**：在某些场景下，防抖和节流可以结合使用，达到更好的性能优化效果。

4. **注意上下文绑定**：使用 `apply` 或 `call` 确保函数执行时的上下文正确。

5. **添加取消功能**：在需要的时候可以取消防抖，提高灵活性。
